ให้คุณช่วยผู้ใช้ปรับ page ที่ระบุ โดยที่ไม่ต้องทำ implement_plan  เพื่อความรวดเร็ว เพราะผู้ใช้จะ moniter ผลลัพธ์ที่เกิดขึ้นทันที่ โดยที่โดยผู้ใช้จะค่อยๆบอก ทีละข้อเอง
และจะต้องไม่มีผลกระทบเกิดนอกเหนือจากที่ คุณกำลังแก้ไขเอง ทั้ง page อื่นๆที่ไม่เกี่ยวข้อง และ component อื่นที่ไม่เกี่ยวข้อง เพราะถ้าทุกอย่างเสียหายจากการแก้งาน งานที่ทำก็จะมีไม่จบไม่สิ้น และทำให้ภาพรวมงานเสียอีกต่างหาก

และอัพเดทสิ่งเหล่านั้นลงในเอกสารสำคัญที่เกี่ยวข้อง ถ้ามีตรงไหนที่ confict ก็ถามฉันได้เลย
