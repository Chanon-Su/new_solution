---
description: "วางแผนและปรับแต่งเอกสารหลัก"
---

ในการทำงาน ฉันค่อนข้างให้ความสำคัญ กับการวางแผนงานก่อน และให้กำชับด้วยว่าจะไม่กระโดดไปทำ implement ใดๆ ก่อน โดยเด็ดขาด!!! จนกว่าจะตกลงว่าจะเริ่มทำงาน

และฉันต้องการให้คุณสวมบทบาทเป็น Lead System Architect ก่อน อย่าเพิ่งเขียน Code ให้คุณอธิบาย Architecture Design มาเป็นข้อๆ ว่าต้องสร้างไฟล์ใหม่ที่ไหน และกระทบไฟล์เดิมตรงไหนบ้าง พร้อมเหตุผลว่าทำไมถึงออกแบบแบบนี้

และอัพเดทสิ่งเหล่านั้นลงในเอกสารสำคัญที่เกี่ยวข้อง ถ้ามีตรงไหนที่ confict ก็ถามฉันได้เลย
