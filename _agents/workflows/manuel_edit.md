---
description: "เมื่อ agent ไม่สามารถแก้ไขบางไฟล์เองได้ ฉันจะ copy-past เอง"
---

ถ้าก่อนหน้านี่มีงานอะไรที่ทำอยู่หรือเตรียมอยู่ ขอให้หยุดการเริ่ม รันหรือการกระทำอะไรที่ต้องใช้ไฟล์นี้เอาไว้ก่อน ไม่อย่างงั้น งานไฟล์ที่ ฉันกำลังทำอยู่ ก็คงจะไม่ได้ถูกใช้กับงาน และ งานนั้นก็จะไม่สมบูรณ์

1. ฉันจะใช้คำสั่งนี้ เมื่อเกิด ไฟล์บางไฟล์ที่ไม่สามารถแก้ไขได้ จะด้วยเหตุผลใดก็ตามไม่สำคัญ
2. ให้คุณตรวจจับว่า ไฟล์ไหน ในงานล่าสุดที่ยังไม่ได้รับการแก้ไข
3. ให้สรา้งเนื้อหาในไฟล์นั้นออกมาเลย แล้วฉันจะเป็นคน Copy-Paste ในไฟล์นั้นเอง
4. ทำเฉพาะไฟล์ที่ติดปัญหาเท่านั้น นอกเหนือจากนั้น คุณเป็นคน edit ตามปกติเหมือนเดิม
5. อย่าลืม กลับไปดูที่ไฟล์ตัวปัญหาด้วย ว่า ข้อมูลที่ฉันเพิ่มไป เนื้อหาครบถ้วนมั้ย? 
6. ถ้าไม่ครบก็แจ้งฉันได้เลย ถ้าดูเนื้อหาข้างในไม่ได้อยู่ดี ก็แค่รายงานว่าดูไม่ได้ และไปทำอย่างอื่นต่อได้เลย
7. ทวงถามฉันว่างานที่ฉันแก้อยู่ แก้เสร็จยัง? ถ้าเสร็จก็เริ่ม กระบวนการที่ทำอยู่ต่อได้เลย ถ้าไม่ก็ hold ไว้ก่อน
8. สุดท้าย หลังแจ้งเสร็จ กลับ run อย่างอื่นต่อด้วยตัวเองได้เลย
